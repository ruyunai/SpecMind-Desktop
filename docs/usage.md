# SpecMind Desktop - 使用说明

## 快速开始

```
1. 启动 SpecMindDesktop.exe
2. Ctrl+, → 输入 DeepSeek API Key → 保存
3. 在需求输入框粘贴客户需求
4. 点击「开始执行」
5. 等待 60-90 秒，查看完整报告
```

## 7 步工作流说明

### 1. 需求输入

在「需求输入」Tab 的文本框中粘贴客户的原始需求（可以是 Word/PDF 粘贴，或直接打字）。

**示例输入**:
```
客户：智联慧学教育科技
需求：
1. 在线课程管理平台，支持课程创建、分类、搜索
2. 排课系统，支持拖拽式排课，冲突自动检测
3. 在线直播教学，支持 1080P，延迟 < 2s
4. 作业批改系统，支持文本/图片/语音批注
5. 学习进度追踪
6. 管理后台数据看板
7. 微信+支付宝双通道支付
8. 需要支持 10 万用户同时在线
9. 要求提供全部源代码
10. 承诺终身免费升级维护
```

### 2. SAR Agent — 需求清洗

清洗脏需求，对齐企业标准能力库，标注过度承诺风险。

**输出**：
- 清洗后的标准化需求
- 过度承诺风险列表

### 3. Legal Agent — 合规预检

检索法规库，评估合规风险，标注违法/违规条款。

**输出**：
- 风险等级（高/中/低）
- 命中法条与建议
- 是否阻断流程

> ⚠ 本节点为辅助预检工具，输出不构成正式法律意见。

### 4. PM Agent — PRD 生成

将清洗后的需求拆解为标准化 PRD（8 模块）+ 功能点标注。

**输出**：
- 背景目标 / 用户故事 / 功能列表 / In/Out 范围 / 验收标准 / 非功能需求 / 埋点要求 / 风险章节
- 功能点列表（标注：标准功能 / 定制功能 / 暂不支持）

### 5. Commercial Agent — 报价生成

基于功能点数量和成本参数，动态计算双报价。

**输出**：
- 标准版报价（含全部 STANDARD + CUSTOM 功能）
- 裁剪版报价（含核心 STANDARD 功能）

**成本参数**（Ctrl+, → 成本参数 Tab 可调）：
| 参数 | 默认值 | 说明 |
|------|--------|------|
| 人天费率 | 3000 | 元/人天 |
| 标准功能工期 | 15 | 每个标准功能所需人天 |
| 定制功能倍率 | 2.0x | 定制功能人天 = 标准 × 倍率 |
| 毛利率 | 40% | |
| 维护费率 | 10% | 维护费 = 开发费 × 比例 |
| 项目周期 | 3 | 月 |

### 6. Contract Agent — 合同条款比对

检索合同模板库，比对 PRD 与标准合同模板的条款冲突。  
（当前为占位节点，需合同草案输入方可完整使用。）

### 7. Review Agent — 三维评审

技术/设计/质量三维评审。

**输出**：
- 技术评审意见（tech）
- 设计评审意见（design）
- 质量评审意见（qa）
- 通过/不通过标记

### 8. Planner Agent — 交付计划

生成 4-6 个阶段的交付计划。

**输出**：
- 阶段名称 + 耗时（周） + 交付物 + 里程碑

## 模型配置

### 打开配置

- 菜单：设置 → 模型配置
- 快捷键：`Ctrl+,`

### 配置项

| 项目 | 说明 |
|------|------|
| 全局 API Key | DeepSeek API Key（加密存储） |
| 全局 Base URL | 默认为 `https://api.deepseek.com/v1` |
| 每 Agent 模型 | 可为每个 Agent 独立配置模型 |

### 预设路由

- **混合路由**：所有 Agent 使用 `deepseek-chat`
- **全 DeepSeek-R1**：切换为推理模型（更慢但更精确）
- **全硅基 GLM-4**：切换为硅基流动（需要硅基 API Key）

---

## 资产库

左边栏「资产库」存放向量化的企业知识：

| 类别 | 内容 | 用途 |
|------|------|------|
| 法规库 | 法律法规条款 | Legal Agent 合规预检 |
| 标准功能库 | 企业标准化产品能力 | SAR Agent 需求对齐 |
| 合同模板库 | 标准合同条款 | Contract Agent 条款比对 |

### 操作

- **搜索**：顶部搜索框输入关键词，实时过滤
- **详情**：点击条目查看完整内容（含来源/条款号/模块名）
- 知识库内容在源码 `seed_data/` 目录中，可编辑后重新运行注入

## 输出导出

执行完成后，可在工作区切换 Tab 查看各 Agent 的输出：

| Tab | 内容 |
|-----|------|
| 需求输入 | 原始需求 + SAR 清洗报告 + 风险列表 |
| PRD | PRD 8 模块 + 功能点标注 + 报价 |
| 附件 | 合同冲突 + 评审意见 + 交付计划 |

---

## 日志

- **实时日志**：右栏底部实时显示执行日志
- **文件日志**：`data/logs/specmind.log`（文本）和 `specmind.jsonl`（JSON 格式）
- 日志自动轮转（5MB × 3 份备份）

---

## 数据安全

| 数据 | 存储位置 | 加密 |
|------|---------|------|
| API Key | `config/secrets.enc` | Fernet（PBKDF2 派生密钥） |
| 密钥种子 | `config/.keyseed` | 机器绑定 |
| 运行日志 | `data/specmind.db` | 明文（审计用） |
| 知识库 | `data/chroma/` | 明文（向量索引） |

## 常见操作场景

### 场景 1：评估新客户需求

```
1. 粘贴客户需求 → 执行
2. 查看 PRD 确认功能范围
3. 查看报价告知客户预算
4. 查看 Legal 确认合规风险
5. 以 PRD + 报价 + 交付计划三份产出交付客户
```

### 场景 2：调整报价参数

```
1. Ctrl+, → 成本参数 Tab
2. 调整人天费率（如从 3000 调为 5000）
3. 点击保存
4. 重新执行 → 报价自动更新
```

### 场景 3：U 盘带到客户现场演示

```
1. 将 SpecMindDesktop/ 文件夹复制到 U 盘
2. 确保 portable.dat 存在
3. 到客户现场插入 U 盘 → 双击 exe
4. 粘贴客户需求 → 现场生成 PRD 和报价
```
