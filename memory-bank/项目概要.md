# 项目概要

## 项目名称
SpecMind Desktop

## 项目类型
ToB 企业级本地 Windows 桌面应用（单文件 exe）

## 核心目标
将传统 PRD 产出流程从 4 小时压缩至 15 分钟，评审一次通过率提升至 75%。
解决三大痛点：PRD 撰写慢、评审反复打回、落地时需求变更。

## 核心需求
1. **7 Agent 协同**：SAR / Legal / PM / Commercial / Contract / Tech-Lead+Design+QA / Planner
2. **标准化 PRD 模板**：8 个强制模块（背景目标、用户故事、功能列表、In/Out 范围、验收标准、非功能需求、埋点要求、风险章节）
3. **全链路审计**：Audit 模块存储所有 State 快照，支持版本回溯
4. **强约束规则**：Legal 高风险直接阻断 PRD 生成；高风险操作必须 Interrupt 人工确认
5. **本地优先**：不接入云端协作，所有数据本地存储，适配数据隐私敏感的 ToB 场景

## MVP 范围
- 三栏 GUI（资产库 / 工作区三 Tab / LangGraph 画布）
- 7 Agent LangGraph 工作流（含 Interrupt 节点）
- 脏数据清洗 → PRD 生成 → 配套交付物全链路
- Audit 模块全链路快照与回溯
- Fernet 加密 API Key

## 成功标准（验收）
| 指标 | 目标值 |
|------|--------|
| exe 双击启动耗时 | ≤ 3 秒 |
| 脏数据 → 清洗报告 | ≤ 30 秒 |
| PRD 生成总耗时 | ≤ 15 分钟 |
| Legal 合规预检与专业法务一致率 | ≥ 85% |
| Audit 全链路回溯 | 任意项目完整可回放 |
| 无 Python 环境 Windows 运行 | 正常 |

## 明确不做
- 不做独立销售工具
- 不做法务正式意见工具（Legal 仅辅助预检）
- 不接入云端协作
