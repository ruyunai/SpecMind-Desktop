# 技术环境

## 技术栈表

| 层级 | 技术 | 版本 | 用途 |
|------|------|------|------|
| 桌面 GUI | PySide6 | 6.11.1 (abi3) | 三栏 QMainWindow 布局、QTabWidget、QTreeView、模型配置对话框 |
| Agent 编排 | langgraph | 0.2.x | StateGraph + Interrupt + 条件边 |
| Checkpoint | langgraph-checkpoint-sqlite | 3.1.0 | LangGraph 持久化（SqliteSaver），修复并行节点 audit_snapshots 丢失 |
| LLM 抽象 | langchain-openai | 0.1.x | OpenAI compatible 接入硅基流动 |
| LLM 路由 | 自研 router | - | GLM-4-Flash / Qwen2.5-72B / DeepSeek-V3 混合路由 |
| 向量库 | chromadb | 1.5.9 | 法规库/合同模板/历史PRD 向量检索（0.5.x 的 hnswlib 无 3.14 wheel，升级到 1.5.9） |
| 嵌入模型 | 硅基流动 embedding | bge-m3 | ChromaDB 默认 embedder |
| 关系库 | sqlite3 | Python 标准库 | 审计日志 + 结构化资产（功能清单/成本模型）+ FTS5 BM25（trigram 分词，支持中文短语匹配） |
| 加密 | cryptography | 43.x | Fernet 对称加密 API Key |
| Word 解析 | python-docx | 1.1.x | SAR 导入 .docx |
| PDF 解析 | PyPDF2 | 3.0.x | SAR 导入 .pdf |
| 打包 | pyinstaller | 6.21.0 | 单文件 exe (--onefile via SpecMindDesktop.spec) |
| Python | CPython | 3.14.x | PySide6 6.11 abi3 wheel 支持 3.10~3.14 |

## 模型路由策略

| Agent | 模型 | 理由 |
|-------|------|------|
| SAR（需求清洗） | GLM-4-Flash | 轻量任务，控成本，响应快 |
| Legal（合规预检） | Qwen2.5-72B-Instruct | 中文合规场景准确率高 |
| PM（PRD 生成） | Qwen2.5-72B-Instruct | PRD 质量是核心，需强模型 |
| Commercial（报价） | DeepSeek-V3 | 推理强，成本计算逻辑复杂 |
| Contract（合同比对） | Qwen2.5-72B-Instruct | 条款冲突识别需强中文理解 |
| Tech/Design/QA（评审） | DeepSeek-V3 | 推理强，多维评审 |
| Planner（交付计划） | GLM-4-Flash | 结构化输出，控成本 |

## 构建与部署命令
```powershell
# 开发环境
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
python src/main.py

# 打包 exe（使用 .spec 文件，含 130 hiddenimports + frozen 路径适配）
pyinstaller --noconfirm SpecMindDesktop.spec

# 输出: dist/SpecMindDesktop.exe（98.5MB 单文件，无控制台）

# frozen 模式数据目录
%APPDATA%/SpecMindDesktop/
  ├── data/          # chroma/sqlite（get_data_dir() 自动切换）
  └── config/        # app.json + secrets.enc
```

## 关键版本兼容性说明
- **Python 3.14**：PySide6 6.11.1 通过 abi3 wheel（cp310-abi3）支持 3.10~3.14，已验证导入正常
- **LangGraph 0.2.x**：Interrupt API 稳定版，0.1.x 的 interrupt 接口不同
- **ChromaDB 1.5.9**：0.5.x 的依赖 chroma-hnswlib==0.7.6 无 Python 3.14 wheel，升级到 1.5.9 解决
- **PySide6 6.11.1**：使用稳定 ABI，一个 wheel 兼容多版本 Python，非 LTS 但功能完整
- **模型配置可视化**：7 个 Agent 的 LLM 配置（API Key/Base URL/模型）通过 GUI 可视化输入切换，支持预设路由一键应用
- **langgraph-checkpoint-sqlite 3.1.0**：修复并行节点 audit_snapshots 丢失问题（0.2.x 版本 checkpointer 在 fan-out/fan-in 场景下 reducer 不完整）
- **FTS5 trigram 分词**：从 unicode61 迁移到 trigram（3 字符滑窗），解决中文短语匹配失败问题；长查询用 4 字符滑窗 OR 合并
